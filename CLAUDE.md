## Технические решения

### Стек
- **Frontend**: Astro 5 + Tailwind CSS 4 + TypeScript
- **Хостинг**: GitHub Pages (статика)
- **i18n**: EN/RU с автоопределением по браузеру

### Архитектурное решение: разделение репозиториев
GitHub Pages на бесплатном плане требует **публичный репозиторий**.
Поэтому лендинг (stalcraft-info) и бэкенд/аналитика должны быть в **разных репозиториях**:
- `stalcraft-info` — публичный, только лендинг (GitHub Pages)
- `stalcraft-backend` — приватный, API/аналитика (когда появится)

### Архитектура (планируется)
- **Аналитика**: PostgreSQL (транзакции) + ClickHouse (time-series цен)
- **API**: Stalcraft API (eapi.stalcraft.net)
  - Приложение: **stalcinfo** (название без "stalcraft" из-за ограничений)
  - Статус: заявка подана (2026-01-04)
