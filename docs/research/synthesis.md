## Executive Summary

### Ситуация на рынке

**Насыщенность:** 5+ активных конкурентов в нише STALCRAFT:
- stalcraft-monitor.ru — лидер (мониторинг аукциона + мобильное приложение)
- stcx.ru — бесплатный мониторинг
- sctools.tech — калькуляторы и сборки
- stalcraftdata.com — комплексная платформа (API заблокирован EXBO)
- stalcrafthq.com — кланы, игроки, выбросы

**Ключевой риск:** stalcraftdata.com уже столкнулся с отказом в доступе к API от EXBO. Та же проблема может повториться для stalcraft.info.

### Монетизация конкурентов

| Сервис | Модель | Цена Premium |
|--------|--------|--------------|
| stalcraft-monitor.ru | Freemium (?) | Не публикуется |
| sctools.tech | Freemium (?) | Не публикуется |
| stcx.ru | Донаты | Бесплатно |
| stalcrafthq.com | Patreon | Бесплатно |
| stalcraftdata.com | Бесплатно | — |

## Рекомендуемая стратегия

### Этап 1: MVP (0-6 месяцев)

**Модель:** Полностью бесплатно

**Цели:**
- Получить доступ к Stalcraft API
- Набрать 5,000+ MAU
- Валидировать востребованность фич

**Фокус:**
- Мониторинг аукциона (если API одобрят)
- Калькуляторы (не требуют API)
- Community building (Discord)

**Расходы:** ~$20-50/мес (хостинг)

### Этап 2: Community (6-12 месяцев)

**Модель:** Patreon/Boosty

**Тарифы:**
- Tier 1: 100 руб/мес — Discord роль, early access
- Tier 2: 300 руб/мес — contributor badge, приоритетный саппорт
- Tier 3: 500 руб/мес — участие в roadmap, beta-тестирование

**Цели:**
- 20-50 supporters
- Покрытие расходов на хостинг
- Валидация willingness to pay

**Ожидаемый доход:** 2,000-15,000 руб/мес

### Этап 3: Freemium (12-18 месяцев)

**Условие:** Только после проверки ToS Stalcraft на монетизацию!

**Free tier:**
- Цены на аукционе (обновление каждые 30 мин)
- История за 7 дней
- 5 предметов в watchlist
- Базовые калькуляторы

**Premium (200-300 руб/мес или 2000 руб/год):**
- Real-time данные (1-5 мин)
- Полная история (12 мес)
- Unlimited watchlist + alerts (Telegram/Email)
- API access
- Приоритетная поддержка

**Ожидаемые метрики:**
- 10,000 MAU
- 2-3% конверсия = 200-300 подписчиков
- Доход: 40,000-90,000 руб/мес

## Заключение

**Позиционирование stalcraft.info:**

> Прозрачный, современный инструмент аналитики STALCRAFT с открытыми ценами и фокусом на community.

1. Прозрачность (цены публикуются открыто)

4. Мультиязычность
