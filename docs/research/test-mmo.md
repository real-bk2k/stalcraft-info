# MMO Analytics Tools - Brief Overview

Краткий список инструментов для аналитики MMO-игр:

## 1. **NewWorld-Companion**
Платформа для отслеживания рынка New World. Показывает цены на сервере, графики изменений, калькулятор крафта.

## 2. **EVE Online Market Data**
API и сайты (например, EVE Marketer, Evepraisal) для анализа рынка EVE Online. Real-time данные по станциям и регионам.

## 3. **Albion Online Data Project**
Открытая платформа сбора данных о ценах в Albion Online. Краудсорсинговый подход через клиентский мод.

## 4. **GW2BLTC / GW2Efficiency**
Инструменты для Guild Wars 2: отслеживание цен на Trading Post, калькуляторы прибыли, портфолио игрока.

## 5. **WoWhead / Undermine Journal**
Аналитика аукционов World of Warcraft: история цен, тренды, поиск выгодных сделок.
