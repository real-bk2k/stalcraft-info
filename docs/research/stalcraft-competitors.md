# Исследование конкурентов STALCRAFT.info

**Дата:** 2026-01-04
**Статус:** Детальный анализ конкурирующих сервисов

---

## Обзор

Исследование охватывает основные сервисы аналитики и инструментов для игры STALCRAFT. Все конкуренты предоставляют различные комбинации функционала: от мониторинга аукциона до калькуляторов сборок.

**Ключевые выводы:**
- Большинство сервисов **бесплатные** с добровольными донатами
- Основной функционал: мониторинг аукциона, калькуляторы, профили игроков
- Точные цены премиум-подписок **не публикуются открыто**
- Главные конкуренты: stalcraft-monitor.ru, sctools.tech, stcx.ru

## Анализ монетизации

### Модели монетизации конкурентов:

1. **Полностью бесплатные с донатами:**
   - stcx.ru (DonationAlerts)
   - stalcrafthq.com (Patreon)
   - stalcraftdata.com (без монетизации)

2. **Freemium (предположительно):**
   - stalcraft-monitor.ru (есть упоминание Premium)
   - sctools.tech (есть премиум-роль)

3. **Проблема прозрачности:**
   - Ни один сервис **не публикует цены открыто**
   - Детали Premium-функций **не раскрываются**
   - Нет публичных прайс-листов

## Выводы для stalcraft.info

### Сильные стороны конкурентов:

1. **stalcraft-monitor.ru:**
   - Комплексный мониторинг аукциона
   - Мобильное приложение
   - Профили игроков

2. **sctools.tech:**
   - Богатый набор калькуляторов
   - Публичные сборки от сообщества
   - Удобный UX

3. **stalcraftdata.com:**
   - Широкий набор инструментов
   - Полностью бесплатный

### Возможности для stalcraft.info:

1. **Прозрачность монетизации:**
   - Публиковать цены открыто
   - Чёткое разделение Free/Premium

2. **Уникальные функции:**
   - Продвинутая аналитика аукциона (PostgreSQL + ClickHouse)
   - API для разработчиков
   - Экспорт данных

3. **Технологическое преимущество:**
   - Современный стек (Astro 5 + Tailwind 4)
   - Быстрая статика (GitHub Pages)
   - Потенциал масштабирования

### Риски:

1. **Насыщенность рынка:**
   - Минимум 5 активных конкурентов
   - Большинство функций уже реализованы

2. **Барьер входа:**
   - Ожидание доступа к STALCRAFT API
   - stalcraftdata.com уже столкнулся с этой проблемой

3. **Монетизация:**
   - Пользователи привыкли к бесплатным сервисам
   - Непонятно, как конкуренты зарабатывают
